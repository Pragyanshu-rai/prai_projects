About Me


Hey there! I am a B.Tech graduate. Through my graduation, I had problems testing my algorithms for OS, Stack, BinaryTree... as It was difficult to test the algorithm against each test as I had to solve it first on paper which took a lot of time, and I had to check it twice to be sure and a lot of times when I needed the solution I had to work on it again and again on paper.

Hence I developed this Web Application so that it could help a lot of students or developers.
